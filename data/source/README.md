# Source Materials

Эта директория для исходных материалов от преподавателя.

## Поддерживаемые форматы:
- PDF (.pdf)
- Word документы (.docx)
- Изображения со словами (.jpg, .png)

## Workflow:

1. **Положи файлы сюда** - просто скопируй PDF/DOCX от препода
2. **Скажи Claude** - "Извлеки слова из data/source/lesson-1.pdf"
3. **Claude создаст Excel** - в `data/processed/` с типами слов
4. **Проверь и скорректируй** - открой Excel, исправь если нужно
5. **Импортируй в бота** - через команду `/import` или запусти бота с файлом

## Примеры файлов:
- `lesson-1-vocabulary.pdf` - словарь урока 1
- `verbs-conjugation.pdf` - таблица спряжений
- `common-phrases.docx` - распространенные фразы

## Примечание:
Файлы в этой директории НЕ коммитятся в git (добавлены в .gitignore).
