# Processed Data

Эта директория для обработанных данных готовых к импорту в бота.

## Формат файлов:

Все файлы должны быть в формате Excel (.xlsx), CSV (.csv) или JSON (.json) с колонками:

| Greek      | Russian        | Type        |
|------------|----------------|-------------|
| κάνω       | делать         | verb        |
| νερό       | вода           | noun        |
| καλός      | хороший        | adjective   |

## Типы слов (Type):
- `noun` - существительное
- `verb` - глагол
- `adjective` - прилагательное
- `adverb` - наречие
- `pronoun` - местоимение
- `preposition` - предлог
- `conjunction` - союз
- `phrase` - фраза

## Workflow:

1. **Claude создает файл** - после обработки PDF из `data/source/`
2. **Ты проверяешь** - открываешь Excel, корректируешь типы/переводы
3. **Импортируешь в бота** - команда `/import` + файл

## Примечание:
Файлы в этой директории НЕ коммитятся в git (добавлены в .gitignore).
Они используются только для локального импорта.
